# Product Image Validation Guide
## Script for Identifying Correct vs Wrong Images

---

## Overview

This guide helps validate whether product images match their product names. Following Amazon's 2026 guidelines, images must accurately represent the product.

---

## Core Validation Rules

### ✅ CORRECT IMAGE Criteria

| Rule | Description |
|------|-------------|
| **Product Match** | Image shows exactly the product in the title |
| **Clean Background** | Pure white (#FFFFFF) or neutral background |
| **Good Lighting** | Well-lit, no shadows or overexposure |
| **Proper Angle** | Shows key features clearly |
| **High Resolution** | Minimum 1000px, recommended 2000px+ |
| **No Text/Watermarks** | Clean, no promotional text overlay |
| **Accurate Color** | True-to-life colors, not artificially enhanced |

### ❌ WRONG IMAGE Indicators

| Rule | Description |
|------|-------------|
| **Mismatched Product** | Different product than described |
| **Stock Photo** | Generic image not specific to product |
| **Wrong Color/Size** | Shows variant not matching listing |
| **Packaging Only** | Box/container without product |
| **Low Quality** | Blurry, pixelated, or dark |
| **Watermarks/Logos** | Third-party branding visible |
| **Distorted** | Heavily filtered or edited |
| **Multiple Products** | Shows bundle not in listing |

## Quick Validation Checklist

### Before Accepting an Image:

```
□ Does the image show the exact product?
□ Is the brand/logo correct?
□ Is the color/pattern accurate?
□ Is the size/proportion right?
□ Is the image high quality (not blurry)?
□ Is the background clean (white/preferred)?
□ Are there no watermarks/logos?
□ Is there no misleading text?
□ Does it match all product variants?
□ Is the angle appropriate?
```

### Red Flags to Reject:

```
□ Stock photo that could be anything
□ Wrong product category entirely
□ Generic placeholder image
□ Heavily filtered/color-shifted
□ Shows multiple products not in listing
□ Product with obvious damage
□ Professional studio photo vs product photo
□ Image from different listing
```

## Automated Validation Script

### Python Script for Batch Validation

```python
#!/usr/bin/env python3
"""
Product Image Validator
Validates if product images match their names/descriptions
"""

import os
import json
import re
from typing import Dict, List, Tuple
from dataclasses import dataclass
from pathlib import Path

@dataclass
class Product:
    id: str
    name: str
    brand: str
    category: str

@dataclass
class ValidationResult:
    product_id: str
    is_valid: bool
    confidence: float
    issues: List[str]
    suggestions: List[str]

class ProductImageValidator:
    
    # Keywords that indicate correct product type
    PRODUCT_TYPE_KEYWORDS = {
        'Electronics': {
            'earbuds': ['earbuds', 'earphone', 'in-ear', 'true wireless'],
            'headphones': ['headphone', 'over-ear', 'on-ear', 'headset'],
            'smartwatch': ['smartwatch', 'watch', 'fitness tracker', 'wearable'],
            'phone': ['smartphone', 'phone', 'mobile', '5g', '4g'],
            'tablet': ['tablet', 'ipad', 'tab'],
            'speaker': ['speaker', 'bluetooth speaker', 'portable speaker'],
            'charger': ['charger', 'adapter', 'power adapter', 'gan'],
            'mouse': ['mouse', 'wireless mouse', 'mouse'],
            'laptop': ['laptop', 'notebook', 'computer'],
        },
        'Fashion': {
            'sneakers': ['sneaker', 'shoe', 'running shoe', 'sport shoe'],
            'jeans': ['jean', 'denim', 'pants'],
            'shirt': ['shirt', 't-shirt', 'tshirt', 'casual shirt'],
            'jacket': ['jacket', 'coat', 'hoodie'],
            'wallet': ['wallet', 'purse', 'card holder'],
            'watch': ['watch', 'wristwatch', 'timepiece'],
            'sunglasses': ['sunglass', 'spectacles', 'eyewear'],
        },
        'Home & Kitchen': {
            'cooker': ['cooker', 'pressure cooker', 'instant pot', 'electric cooker'],
            'purifier': ['purifier', 'air cleaner', 'hepa'],
            'vacuum': ['vacuum', 'vacuum cleaner', 'vacuum cleaner'],
            'fan': ['fan', 'ceiling fan', 'table fan', 'bladc'],
            'flask': ['flask', 'thermos', 'water bottle', 'insulated bottle'],
        },
        'Health & Fitness': {
            'yoga mat': ['yoga mat', 'exercise mat', 'fitness mat'],
            'protein': ['protein', 'whey', 'supplement', 'nutrition'],
            'massager': ['massager', 'massage gun', 'percussion'],
            'racket': ['racket', 'badminton', 'squash', 'tennis'],
        },
        'Beauty': {
            'serum': ['serum', 'essence', 'treatment'],
            'foundation': ['foundation', 'makeup', 'concealer'],
            'facewash': ['facewash', 'face wash', 'cleanser'],
            'straightener': ['straightener', 'flat iron', 'hair straightener'],
        },
        'Books': {
            'book': ['book', 'paperback', 'hardcover', 'novel'],
        }
    }
    
    # Keywords that indicate WRONG image
    NEGATIVE_KEYWORDS = {
        'Electronics': ['cable', 'case', 'cover', 'stand', 'mount', 'protector'],
        'Fashion': ['box', 'bag', 'accessory', 'jewelry'],
        'Home': ['furniture', 'decor', 'curtain', 'carpet'],
        'Books': ['kindle', 'ebook', 'audio', 'summary'],
    }
    
    def __init__(self, products: List[Product]):
        self.products = products
        
    def validate_product_image(self, product: Product, image_url: str = None, 
                             image_filename: str = None) -> ValidationResult:
        """Validate if image matches product"""
        
        issues = []
        suggestions = []
        
        # Check product name contains expected keywords
        name_lower = product.name.lower()
        brand_lower = product.brand.lower()
        
        # Get category keywords
        category = product.category
        found_correct_type = False
        confidence = 0.5
        
        if category in self.PRODUCT_TYPE_KEYWORDS:
            for type_name, keywords in self.PRODUCT_TYPE_KEYWORDS[category].items():
                for keyword in keywords:
                    if keyword.lower() in name_lower:
                        found_correct_type = True
                        confidence += 0.15
                        
        # Check for negative indicators
        if category in self.NEGATIVE_KEYWORDS:
            for neg_keyword in self.NEGATIVE_KEYWORDS[category]:
                if neg_keyword in name_lower:
                    issues.append(f"Name contains accessory term: {neg_keyword}")
                    confidence -= 0.1
                    
        # Brand check
        if brand_lower in name_lower:
            confidence += 0.1
        else:
            suggestions.append("Consider including brand in image filename")
            
        # Image filename validation (if provided)
        if image_filename:
            filename_lower = image_filename.lower()
            
            # Check for product type in filename
            if category in self.PRODUCT_TYPE_KEYWORDS:
                type_found = False
                for type_name, keywords in self.PRODUCT_TYPE_KEYWORDS[category].items():
                    for keyword in keywords:
                        if keyword.lower() in filename_lower:
                            type_found = True
                            confidence += 0.1
                            
                if not type_found:
                    suggestions.append(f"Filename should contain product type from: {list(self.PRODUCT_TYPE_KEYWORDS[category].keys())}")
                    
            # Check for brand in filename
            if brand_lower in filename_lower:
                confidence += 0.1
                
        # Determine validity
        is_valid = found_correct_type and confidence >= 0.6
        
        if not found_correct_type:
            issues.append(f"Could not identify product type from name: {product.name}")
            
        if confidence < 0.6:
            issues.append(f"Low confidence score: {confidence:.2f}")
            
        return ValidationResult(
            product_id=product.id,
            is_valid=is_valid,
            confidence=min(1.0, confidence),
            issues=issues,
            suggestions=suggestions
        )
    
    def validate_catalog(self, catalog_data: List[dict]) -> Dict:
        """Validate entire product catalog"""
        
        results = []
        valid_count = 0
        
        for item in catalog_data:
            product = Product(
                id=item.get('id', ''),
                name=item.get('name', ''),
                brand=item.get('brand', ''),
                category=item.get('category', '')
            )
            
            result = self.validate_product_image(product)
            results.append(result)
            
            if result.is_valid:
                valid_count += 1
                
        return {
            'total_products': len(catalog_data),
            'valid_products': valid_count,
            'invalid_products': len(catalog_data) - valid_count,
            'validation_rate': valid_count / len(catalog_data) if catalog_data else 0,
            'results': results
        }

# Example Usage
if __name__ == "__main__":
    # Sample products
    products = [
        {"id": "1", "name": "Apple AirPods Pro 2nd Gen", "brand": "Apple", "category": "Electronics"},
        {"id": "2", "name": "Samsung Galaxy Watch 6 Classic", "brand": "Samsung", "category": "Electronics"},
        {"id": "3", "name": "Nike Air Max 270 React", "brand": "Nike", "category": "Fashion"},
    ]
    
    validator = ProductImageValidator([])
    results = validator.validate_catalog(products)
    
    print(f"Total: {results['total_products']}")
    print(f"Valid: {results['valid_products']}")
    print(f"Invalid: {results['invalid_products']}")
    print(f"Validation Rate: {results['validation_rate']:.1%}")
```

## Image Naming Convention

### Recommended Filename Format

```
{Brand}-{ProductName}-{Variant}-{Angle}.{ext}

Examples:
├── apple-airpods-pro-2nd-gen-front-white.jpg
├── samsung-galaxy-watch6-classic-side-black.jpg
├── nike-air-max-270-react-front-blue.jpg
├── sony-wh1000xm5-headphones-front-black.jpg
├── logitech-mx-master-3s-mouse-front-graphite.jpg
├── instant-pot-duo-7in1-cooker-front-silver.jpg
├── atomic-habits-james-clear-cover.jpg
└── minimal-10-niacinamide-serum-front.jpg
```

### Bad Filename Examples

```
❌ img001.jpg
❌ product123.png
❌ photo.jpg
❌ airpods (1).jpeg
❌ samsung-watch.jpg (too vague)
❌ IMG_20240101_123456.jpg
```
